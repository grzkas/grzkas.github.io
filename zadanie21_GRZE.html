Użycie ID w HTML i CSS

Tabela pojęć

Pojęcie Znaczenie pojęcia
Co znaczy skrót ID? ID (ang. identifier) oznacza identyfikator, który pozwala na unikalne oznaczenie elementu w kodzie HTML.
Jak jest cel stosowania ID? ID pozwala na jednoznaczną identyfikację elementu w dokumencie, co ułatwia stylizację oraz obsługę za pomocą JavaScript.
Znaczenie znaku # Znak # w CSS oznacza selektor identyfikatora. Używa się go do odniesienia do konkretnego elementu o danym ID.
Jak następuje wywołanie ID? ID przypisuje się do elementu w HTML za pomocą atrybutu id="nazwa", a następnie w CSS odwołuje się do niego jako #nazwa { właściwości }.
Uwaga na temat stosowania ID ID powinien być unikalny w obrębie całego dokumentu – nie można przypisywać tego samego ID do wielu elementów.
Czy konieczne jest wyliczanie? Nie, ID nie wymaga numerowania, ale jeśli stosujemy wiele podobnych elementów, lepiej używać klas (class) zamiast ID.

Teoria

Identyfikatory pozwalają na przypisanie formatowania do unikalnych elementów.

Stosujemy znak # przed nazwą ID w arkuszu styli.
